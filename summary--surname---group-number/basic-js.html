Value Types

Number
Any numeric value
String
Characters inside quote marks
Boolean
True or False
Null or Undefined
Empty and meanin­gless
Object
Any value associated with the object
Function
Value returned by a function

Data Stuctures

Call Stack (Стек вызовов)
A mechanism for an interpreter to keep track of its place in a script that calls multiple functions
Queue (Очередь Кью)
An element is inserted at the end of the queue and is removed from the front of the queue.
Linked List (Cвязанный список узлов и ссылок или указателей)
это структура данных, в которой несколько значений хранятся линейно
Set (значения)
набор хорошо определенных и обособленных объектов
Hash Table (таблица кэширования)
структура данных, которая строится по принципу ключ-значение
Tree
это многослойная (многоуровневая) структура
Trie (Нагруженное (префиксное) дерево)
Данные в нем сохраняются последовательно — каждый узел дерева представляет собой один шаг.
Graph (график)
представляет собой коллекцию связанных между собой узлов
