Logical Operators

x == y
Returns true if x and y are equal
x === y
Returns true if x and y are identical
x != y
Returns true if x and y are not equal
x !== y
Returns true if x and y are not identical
x > y
Returns true if x is greater than y
x >= y
Returns true if x is greater than or equal to y
x < y
Returns true if x is less than y
x <= y
Returns true if x is less than or equal to y
x && y
Returns true if both x and y are true
x || y
Returns true if either x or y are true
!x
Returns true if x is false

Lodash Library

_.map
Метод map создаёт новый массив, который будет состоять из результатов вызова callback функции для каждого элемента массива, переданного первым аргументом.
_.includes
метод includes проверяет, находится ли значение в коллекции.
_.filter
С помощью метода filter можно отфильтровать объект.
_.find
итерируется по элементам переданного массива и возвращает первый элемент, для которого вызов callback функции вернёт true.
_.groupBy
метод, который позволит сгруппировать данные массивов на основе заданных параметров.
_.difference, _.differenceBy, _.intersection
Метод difference создаёт новый массив с элементами, которых нет в других переданных массивах. Другими словами, он сравнивает два или более массива и возвращает разницу между ними.
_.uniq, _.union
Метод uniq исключает из массива повторяющиеся элементы.
Метод union исключает повторяющиеся элементы из всех переданных массивов, и возвращает новый созданный на этой основе массив.
_.pickBy
Создаёт новый объект с перечисляемыми свойствами переданного объекта, для которых вызов callback функции вернёт true.
_.assign, _.merge
Метод assign присваивает переданному первым аргументом объекту перечисляемые свойства других переданных объектов, как показано на примере.
Метод merge осуществляет глубокое рекурсивное объединение в объект собственных и унаследованных перечисляемых свойств переданных объектов.
_.clone, _.cloneDeep
Эти два метода создают копию переданного объекта, и различаются между собой некоторыми важными особенностями.
_.attempt и _.isError
Метод attempt отлавливает ошибку JSON.parse и вместо этого он возвращает объект Error. Ход выполнения программы не прерывается в случае ошибки JSON.parse, а полученный объект Error можно проанализировать с помощью метода isError.
_.chain
Метод chain создаёт так называемый экземпляр оболочки lodash, с помощью которого можно применить к данным несколько методов.
_.value
Метод value, который завершает цепочку, как раз делает финальный штрих и возвращает готовое обработанное значение.
 

Циклы и итерации

Цикл for
Цикл for повторяет действия, пока не произойдёт какое-либо специальное событие завершения цикла.
Цикл do...while
Цикл do...while повторяется пока заданное условие истинно.
Цикл while
Цикл while выполняет выражения пока условие истинно.
Метка (label)
Метка представляет собой оператор с идентификатором, который позволяет вам ссылаться на какое-то место в вашей программе.
break
Используйте оператор break, чтобы прерывать цикл, переключать управление или в сочетании с оператором метка.
continue
Оператор continue используется, чтобы шагнуть на шаг вперёд в циклах while, do-while, for или перейти к метке.
for...in
Оператор for...in проходит по всем перечислимым свойствам объекта. JavaScript выполнит указанные выражения для каждого отдельного свойства.
for...of
Оператор for...of создаёт цикл, вызывая на каждой итерации функцию с выражениями, которые надо выполнить для получения значения каждого отдельного свойства.
